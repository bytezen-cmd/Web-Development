# Web Development Learning Projects

A collection of beginner-friendly HTML projects designed to teach fundamental web development concepts.

## Projects Included

### 1. CatPhotoApp
An introductory project showcasing HTML essentials including semantic structure, images, links, and lists. This project demonstrates how to organize content with proper HTML tags and create a visually organized webpage about cats.

**Key Concepts:** Semantic HTML, images, hyperlinks, ordered/unordered lists, figures and captions

### 2. Build a Curriculum Outline
A simple structured document that introduces the full stack development curriculum. This project shows how to organize educational content using heading hierarchy and paragraphs.

**Key Concepts:** Heading hierarchy (h1, h2, h3), paragraph organization, basic document structure

### 3. Debug Camperbot's Profile Page
A profile page for Camperbot that introduces debugging concepts. Students learn to identify and fix common HTML issues while building a personal profile page.

**Key Concepts:** Debugging HTML, proper nesting, semantic markup, profile content organization

### 4. Debug a Pet Adoption Page
A pet adoption website that provides real-world practice with debugging. This project combines multiple HTML elements including images, links, and headings in a practical scenario.

**Key Concepts:** Debugging, image integration, internal links, multi-section layout

## Getting Started

### Prerequisites
- A text editor (VS Code, Sublime Text, etc.)
- A modern web browser
- Basic familiarity with HTML

### Installation

```bash
git clone <repository-url>
cd <repository-name>
```

### Running the Projects

Simply open any `.html` file in your web browser to view the project. No server or build tools are required.

## Learning Path

1. Start with **CatPhotoApp** to learn HTML fundamentals
2. Move to **Build a Curriculum Outline** to practice content organization
3. Work through **Debug Camperbot's Profile Page** to develop debugging skills
4. Complete **Debug a Pet Adoption Page** for a real-world scenario
